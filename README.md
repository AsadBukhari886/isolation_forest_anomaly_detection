# IoT Sensor Anomaly Detection Using Isolation Forest

## Author
Asad Bukhari

## Overview
This project demonstrates **anomaly detection** in IoT sensor data using **Isolation Forest**.  
It simulates temperature and humidity sensors, injects anomalies, and detects them using a machine learning model.

## Features
- Synthetic IoT/edge sensor dataset
- Isolation Forest anomaly detection
- Clear visualization of anomalies
- Reproducible results using `random_state=42`

## How to Run
1. Open `Isolation_Forest_Anomaly_detection.ipynb` in [Google Colab]([https://colab.research.google.com/](https://colab.research.google.com/drive/1o07tnw6kBOr5PnDdQbTNXhQJKd5Xc7dz?usp=sharing))
2. Run all cells sequentially
3. View the anomaly plot and summary output
